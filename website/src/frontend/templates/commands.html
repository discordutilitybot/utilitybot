Complete list of UtilityBot commands

Moderation

Command Description
u!kick [member] (reason) Kicks a member.
u!ban [member] (reason) Bans a member.
u!unban [member] (reason) Unbans a member.
u!mute [member] Mutes a member.
u!unmute [member] Unmutes a member.
u!purge [amount] Purges a specified amount of messages.

Fun

Command Description
u!roast [member] Roasts a member.
u!slap [member] Slaps a member.
u!coinflip Flips a coin.

Information

Command Description
u!help General help command.
u!mod For help with the moderation module.
u!fun For help with the fun module.
u!prefix Get the prefix for the bot.
u!invite Get the invite for UtilityBot.
u!avatar [member] Get the avatar of a user.
u!serverinfo Gets the server information.
u!userinfo [member] Get the information about a user.
